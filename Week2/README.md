# 📅 WEEK 2 : 자료구조 심화 & DFS/BFS 응용

**핵심 목표:** "익숙한 자료구조의 낯선 활용법을 익히고, 탐색의 범위를 넓힌다."

### **📅 DAY 1 : 스택 (Stack) - 응용과 문자열 처리**
기본적인 `push/pop` 구현은 졸업했습니다. 이제 스택을 활용해 복잡한 문자열을 처리하거나, $O(N)$의 효율성을 요구하는 문제를 풉니다.

1.  **[백준 17298] 오큰수:** (골드4/★필수) `2493 탑`과 형제 문제입니다. 오른쪽에 있는 수 중 나보다 큰 첫 번째 수를 찾습니다. 이중 반복문을 쓰면 시간 초과가 나므로 **모노톤 스택**을 반드시 써야 합니다.
2.  **[백준 4949] 균형잡힌 세상:** (실버4) 소괄호`()`와 대괄호`[]`가 섞여 나옵니다. 짝이 맞는지 확인할 때 조건 처리가 더 꼼꼼해야 합니다.
3.  **[백준 1935] 후위 표기식2:** (실버3) 컴퓨터가 수식을 계산하는 방식(Postfix)을 이해하고 스택으로 계산기를 직접 구현해 봅니다.
4.  **[백준 2504] 괄호의 값:** (골드5/🌶️도전) 단순히 올바른 괄호인지 묻는 게 아니라, 괄호마다 배점($\times 2, \times 3$)이 다르고 중첩될 때 곱셈 분배 법칙을 코드로 구현해야 합니다. (까다로움 주의)

### **📅 DAY 2 : 큐(Queue) & 덱(Deque) - 시뮬레이션**
단순한 줄 서기가 아닙니다. 원형으로 돌거나 상황에 따라 앞뒤로 넣고 빼는 유연함이 필요합니다.

1.  **[백준 2164] 카드2:** (실버4) 제일 위 카드를 버리고, 그다음 카드를 제일 아래로 옮기는 동작을 반복합니다. `collections.deque`의 기초 체력 훈련.
2.  **[백준 2346] 풍선 터뜨리기:** (실버3) 원형 큐 문제입니다. 풍선 안에 있는 종이의 숫자만큼 왼쪽이나 오른쪽으로 이동해야 합니다. 덱의 `rotate()` 기능을 적극 활용하세요.
3.  **[백준 1021] 회전하는 큐:** (실버3) 원하는 원소를 뽑아내기 위해 왼쪽으로 돌리는 게 빠른지, 오른쪽으로 돌리는 게 빠른지 판단하는 그리디한 큐 문제입니다.
4.  **[프로그래머스 Lv.2] 다리를 지나는 트럭:** (스택/큐) 트럭이 다리 위에 올라가고 내려가는 시간을 정확하게 시뮬레이션해야 합니다. 큐에 `0`을 채워 넣어 다리 길이를 유지하는 테크닉을 익힙니다.

### **📅 DAY 3 : 그래프 탐색(DFS/BFS) - 기초 변형**

1.  **[백준 11724] 연결 요소의 개수:** (실버2) 그래프가 끊겨 있을 때, 총 몇 개의 덩어리(Connected Component)가 있는지 `visited` 배열을 보며 모든 정점을 탐색합니다. [1]
2.  **[백준 4963] 섬의 개수:** (실버2) `상하좌우` 4방향이 아니라, **대각선 포함 8방향**을 탐색해야 합니다. `dx`, `dy` 배열을 8개로 늘려보세요. [1]
3.  **[백준 2644] 촌수계산:** (실버2) 나와 친척 사이가 몇 촌인지 구하는 문제입니다. "최단 거리" 개념이므로 **BFS**로 풀면 촌수(Level)가 바로 나옵니다.
4.  **[백준 1388] 바닥 장식:** (실버4) `-` 모양은 가로로 연결, `|` 모양은 세로로 연결됩니다. DFS 탐색 시 진행 방향 제약을 두는 연습입니다.

### **📅 DAY 4 : BFS 집중 훈련 - 상태가 있는 최단 거리**
단순히 $x, y$ 좌표만 이동하는 게 아니라, 특별한 이동 규칙이 있는 최단 거리 문제입니다.

1.  **[백준 1697] 숨바꼭질:** (실버1) (이미 추천드렸지만 너무 중요해서 유지) 1차원 배열에서의 BFS입니다.
2.  **[백준 7562] 나이트의 이동:** (실버1) 체스 나이트가 $(r, c)$까지 가는 최소 이동 횟수. `dx`, `dy`를 나이트 이동 규칙 `(-2, -1, ...)` 8개로 설정하면 끝입니다.
3.  **[백준 13549] 숨바꼭질 3:** (골드5/★중요) 순간이동($2*X$)은 0초가 걸립니다. 가중치가 0인 간선이 있을 때는 **덱(Deque)을 이용한 0-1 BFS**를 써야 가장 빠릅니다. (큐의 앞에 넣기)

탐색을 하다가 "아니면 되돌아오는" 백트래킹의 기초를 다집니다.

1.  **[백준 15649] N과 M (1):** (실버3) 1부터 N까지 자연수 중 M개를 고르는 수열. 백트래킹의 Hello World입니다. `visited` 처리 후 재귀 호출, 돌아와서 `visited` 해제하는 패턴을 익힙니다.
2.  **[백준 15650] N과 M (2):** (실버3) `15649`에서 **오름차순** 조건이 추가됩니다. 조합(Combination)을 구현하는 문제입니다.
3.  **[백준 10026] 적록색약:** (골드5) (유지) 방문 배열 `visited`를 초기화하고 다시 탐색하는 로직 훈련에 최적입니다.
