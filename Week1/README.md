### 📅 **DAY 1 : 입출력 & 기초 구현 마스터**
입출력 속도, 리스트 컴프리헨션, 기본 내장 함수(`min`, `max`, `sum`)를 손에 익힙니다.

1.  **[백준 2577] 숫자의 개수:** 숫자를 `str()`로 변환해서 `count()` 쓰는 법 익히기.
2.  **[백준 2675] 문자열 반복:** 이중 반복문과 문자열 더하기 패턴.
3.  **[백준 10818] 최소, 최대:** 파이썬의 `min()`, `max()` 함수가 얼마나 편한지 느끼기.
4.  **[백준 1546] 평균:** 점수 조작하기. 리스트의 값을 일괄 변경하는 로직 연습.

### 📅 **DAY 2 : 문자열 다루기 (String Manipulation)**
문자열을 쪼개고, 찾고, 변환하는 것은 모든 기업 코테의 기본입니다.

1.  **[백준 1157] 단어 공부:** (★필수) `upper()`, `set`, 딕셔너리(또는 `Counter`) 활용의 교과서.
2.  **[백준 2908] 상수:** 문자열 슬라이싱 `[::-1]`(뒤집기) 활용법.
3.  **[백준 10809] 알파벳 찾기:** `find()` 함수와 아스키코드(`ord`) 이해하기.
4.  **[백준 11720] 숫자의 합:** 문자열로 된 숫자를 하나씩 떼어서 정수로 변환하기.

검색 속도를 $O(1)$로 줄이는 딕셔너리 연습입니다.

1.  **[백준 10816] 숫자 카드 2:** 리스트로 `count`하면 시간 초과 납니다. **딕셔너리(Hash Map)**로 풀어야 합니다.
2.  **[백준 1764] 듣보잡:** 두 집합의 교집합(`&`)을 구하는 문제. 코드가 얼마나 짧아지는지 확인하세요.
3.  **[프로그래머스 Lv.1] 완주하지 못한 선수:** 해시 문제의 정석.
4.  **[프로그래머스 Lv.1] 폰켓몬:** 중복 제거(`set`)와 논리적 사고.

2.  **[백준 1181] 단어 정렬:** (★중요) 길이 순 정렬 후 사전 순 정렬. `lambda` 정렬(`key=lambda x: (len(x), x)`) 마스터하기.

4.  **[백준 1427] 소트인사이드:** 문자열을 내림차순으로 정렬하는 법.
