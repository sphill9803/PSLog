### 📅 **DAY 1 : 구현 & 완전탐색 (Brute Force)**
"조건이 까다로운 구현"과 "무식하게 다 해보는 탐색"을 연습합니다.

1.  **[백준 1316] 그룹 단어 체커:** (실버5) 이미 나온 문자가 떨어져서 다시 나오면 안 됩니다. 기록(`set`이나 `list`)하면서 푸는 습관.
2.  **[백준 2941] 크로아티아 알파벳:** (실버5) `c=`, `dz=` 같은 두 글자를 한 글자로 치환(`replace`)하는 센스.
3.  **[백준 2563] 색종이:** (실버5/★추천) 100x100 도화지(2차원 배열)에 1을 채워 넣고 `sum`으로 넓이 구하기. **삼성전자 코테의 기초**입니다.
4.  **[백준 1018] 체스판 다시 칠하기:** (실버4/★필수) 8x8로 자를 수 있는 **모든 경우의 수**를 다 해봐야 합니다. 4중 for문의 두려움을 없애세요.

### 📅 **DAY 3 : 해시(Hash) & 집합(Set)**

1.  **[백준 1620] 나는야 포켓몬 마스터 이다솜:** (실버4) 이름으로 번호 찾고, 번호로 이름 찾기. 딕셔너리를 **두 개** 만들어야 빠릅니다.
2.  **[백준 1764] 듣보잡:** (실버4) 두 명단의 교집합(`&`)을 구하세요. `set`을 쓰면 코드가 3줄로 끝납니다.
3.  **[프로그래머스 Lv.2] 전화번호 목록:** (해시) 한 번호가 다른 번호의 접두어인지 확인. 정렬 후 인접 비교하거나 해시를 씁니다.
4.  **[프로그래머스 Lv.2] 의상:** (해시/수학) 경우의 수 공식 `(A+1)*(B+1)*... - 1`을 코드로 구현합니다.

1.  **[백준 1181] 단어 정렬:** (실버5) 길이 순, 그다음 사전 순. `lambda` 정렬(`key=lambda x: (len(x), x)`) 마스터하기.
2.  **[백준 18870] 좌표 압축:** (실버2) 값의 크기가 중요하지 않고 **순위**만 중요할 때 씁니다. `set`과 `dict`의 콜라보.
3.  **[백준 1931] 회의실 배정:** (실버1/★그리디 정석) 끝나는 시간이 빠른 순서대로 회의를 잡는 것이 정답입니다. 증명된 그리디 패턴.
4.  **[백준 11047] 동전 0:** (실버4) 큰 동전부터 거슬러 주는 가장 쉬운 그리디 문제입니다.
